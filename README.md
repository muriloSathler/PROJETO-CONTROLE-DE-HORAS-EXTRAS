Resumo do Projeto:
Este projeto integrador, desenvolvido em parceria com a startup Base27, tem como
objetivo principal criar um sistema digital para controle de horas extras, substituindo
os processos manuais ainda utilizados por muitas empresas. A proposta busca
automatizar etapas como registro, validação e aprovação das horas trabalhadas
além da jornada, resolvendo problemas de lentidão, erros frequentes e dificuldade
de acompanhamento inerentes aos métodos tradicionais controle de horas extras,
substituindo os processos manuais ainda utilizados por muitas empresas. A
proposta busca automatizar etapas como registro, validação e aprovação das horas
trabalhadas além da jornada, resolvendo problemas de lentidão, erros frequentes e
dificuldade de acompanhamento específicos aos métodos tradicionais.
A arquitetura do sistema foi organizada em módulos especializados, começando
pela definição das estruturas de dados fundamentais: usuário para gerenciar as
informações dos funcionários e gestores, e HoraExtra para controlar todo o ciclo de
vida dos registros de horas trabalhadas além da jornada. Foi implementado um
sistema de autenticação que diferencia o acesso entre funcionários e gestores,
onde os primeiros podem registrar e consultar suas próprias horas, enquanto os
gestores têm acesso às funcionalidades de aprovação e relatórios gerais.
Para os funcionários, foram desenvolvidas as funcionalidades de registro de horas
com descrição, cálculo automático e consulta do histórico pessoal. Para os
gestores, foi implementado o sistema de aprovação em lote, visualização de todas
as horas pendentes e geração de relatórios consolidados.
O sistema resultante não apenas resolve o problema prático de controle de horas
extras, mas também serve como material didático valioso para o aprendizado de
algoritmos, estruturas de dados e lógica de programação em C, para evoluir para
linguagens modernas e ambientes corporativos reais.
